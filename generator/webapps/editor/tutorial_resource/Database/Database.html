重要な注意：

  • メディアをインターネット上で閲覧できるようにするには、メディア素材（ムービー、ポスターイメージ）をだれでもアクセスできる公開サーバに置く必要があります。
  • あなたの Web ページ以外の場所にメディア素材を保存して利用する場合は、URL パラメータを変更する必要があります。
  • ムービーの URL パラメータには、ソースムービではなく、必ず参照ムービー（Database.mov）を指定してください。
  • 参照ムービーは、ソースムービーと同じフォルダ内に保管しておく必要があります。
  • ソースムービーの名前は変更できませんが、参照ムービーの名前は変更できます（その場合は、HTML コードでも変更することを忘れないでください）。
  • XHTML として正しく機能させるために、<style> タグと最初の <script> タグを、ページの <head> タグ内に記載してください。
